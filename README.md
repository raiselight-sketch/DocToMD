# DocToMD Converter

모든 문서(.docx, .pdf, .pptx, .xlsx, .hwp 등)를 Markdown으로 일괄 변환하는 Mac/Windows 데스크톱 앱.

## 실행
```bash
source .venv/bin/activate    # Mac
python src/main.py
```
