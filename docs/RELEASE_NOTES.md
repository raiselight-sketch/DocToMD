# 🚀 DocToMD Converter 릴리스 노트

## [v1.1.0] - 2026-05-10

### ✨ 새로운 기능
- **AI 스마트 후처리**: 로컬 Ollama(Gemma4) 또는 Gemini API를 사용하여 변환된 Markdown의 가독성을 지능적으로 개선 (줄바꿈 복구, 오타 교정).
- **HWPX 정식 지원**: ZIP/XML 직접 파싱 방식을 통해 HWPX 문서의 텍스트 추출 기능 구현.
- **Pages 변환 품질 향상**: AppleScript 변환 후 노이즈(인쇄 헤더, 페이지 번호 등) 제거 로직 및 한국어 특화 줄바꿈 복구 알고리즘 적용.
- **설정 관리 시스템**: `config.json`을 통해 출력 경로, AI 모델 설정 등을 영구적으로 저장하고 관리.
- **UI 개선**: AI 후처리 옵션 체크박스 추가 및 사용자 설정 유지 기능 도입.

### 🛠 수정 및 개선
- HWP 변환 시 한국어 조사 뒤에서 줄바꿈이 일어나는 현상 대폭 개선.
- 설정된 출력 경로가 프로그램 재시작 시에도 유지되도록 보정.
