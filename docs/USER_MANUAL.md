# 📄 DocToMD Converter 사용 설명서

본 매뉴얼은 비개발자(목회자, NGO 활동가 등) 사용자를 위한 DocToMD Converter의 기본 사용법을 안내합니다.

---

## 1. 설치 및 실행 방법

### Mac 사용자의 경우
1. 배포된 `DocToMD.dmg` 파일을 실행합니다.
2. `DocToMD.app`을 `Applications` 폴더로 드래그하여 설치합니다.
3. 앱을 실행할 때 "확인되지 않은 개발자" 경고가 뜨면 `우클릭 > 열기`를 선택해 주세요. (최초 1회)

### Windows 사용자의 경우
1. `DocToMD_Setup.exe` 파일을 실행하여 설치를 진행합니다.
2. 바탕화면이나 시작 메뉴의 `DocToMD` 아이콘을 더블 클릭하여 실행합니다.

---

## 2. 주요 기능 및 사용법

### 1) 변환할 파일 추가하기
- 프로그램 중앙의 점선 영역(**Dropzone**)으로 변환하고 싶은 파일이나 폴더를 마우스로 끌어다 놓으세요.
- **지원 파일**: Word(.docx), PDF, PowerPoint(.pptx), Excel(.xlsx), 한글(.hwp), 이미지(.png, .jpg), 텍스트(.txt) 등

### 2) 대기 리스트 관리
- 추가된 파일은 중앙 리스트에 표시됩니다.
- 실수로 추가한 파일은 오른쪽의 `✕` 버튼을 눌러 목록에서 지울 수 있습니다.

### 3) 저장 위치 설정
- 하단의 **[변경]** 버튼을 눌러 변환된 Markdown(.md) 파일이 저장될 폴더를 선택하세요. 기본적으로 바탕화면의 `DocToMD_Result` 폴더에 저장됩니다.

### 4) 변환 시작
- 오른쪽 하단의 **[변환 시작 ▶]** 버튼을 누르면 작업이 시작됩니다.
- 파일별로 `✅ 완료` 또는 `❌ 실패` 표시가 실시간으로 나타납니다.

---

## 3. 결과 확인

작업이 완료되면 설정한 폴더에 다음과 같은 파일들이 생성됩니다.
- **변환된 파일**: 원본과 동일한 이름의 `.md` 파일
- **이미지 폴더**: 문서 내에 이미지가 포함된 경우 별도의 폴더로 추출됩니다.
- **변환 보고서**: `conversion_report_....md` 파일에서 전체 변환 결과를 한눈에 확인할 수 있습니다.

## 4. 자주 묻는 질문 (FAQ)

- **Q: PDF 글자가 인식이 안 돼요.**
  - A: 스캔된 이미지 형태의 PDF인 경우 OCR 기능이 작동합니다. 시스템에 Tesseract OCR이 설치되어 있어야 더욱 정확한 인식이 가능합니다.
- **Q: 한글(.hwp) 파일 표가 깨져요.**
  - A: 현재 버전은 텍스트 중심의 변환을 지원합니다. 복잡한 표는 변환 후 Markdown에서 일부 수정이 필요할 수 있습니다.
- **Q: 실행이 안 돼요.**
  - A: Mac에서는 `시스템 설정 > 보안 및 개인정보 보호`에서 차단된 앱 실행을 허용해 주어야 할 수도 있습니다.
